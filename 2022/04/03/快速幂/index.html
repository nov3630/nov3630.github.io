快速幂

以计算2^10为例，正常的计算需要进行九次乘法计算。时间复杂度为O(n)。为减少计算次数，可以使用快速幂将时间复杂度降为O(logn)。

快速幂的基本思想就是就是将偶数次方拆分成两部分，只需计算其中一部分，将结果保存下来，即可避免进行另一部分的计算。

例如：2^10可以拆分成（2^5）*（2^5），只需计算前面一部分，并将结果保存下来，然后平方，即可得到2^10。实际计算中，使用递归的方式，将需要计算的式子的指数不断拆分，从而将时间复杂度降到O(logn)。

实例（以计算2^10为例）：

#include <bits/stdc++.h>
using namespace std;
typedef long long ll;

ll x, n;

ll calc(ll x, ll n) {
    if (n == 1) return x;
    else if (!(n & 1)) {
        ll temp = calc(x, n / 2);
        return temp * temp;
    }
    else return x * calc(x, n - 1);
}

int main() {
    x = 2; n = 10;
    int ans = calc(x, n);
    printf("%lld\n", ans);
}
def calc(x, n):
    '''
    :param x: 底数
    :param n: 指数
    :return: 使用快速幂的计算结果
    '''
    if n == 1:
        return x
    elif not (n % 2):
        temp = calc(x, n / 2)
        return temp * temp
    else:
        return x * calc(x, n - 1)

if __name__ == '__main__':
    x = 2
    n = 10
    ans = calc(x, n)
    print(ans)
